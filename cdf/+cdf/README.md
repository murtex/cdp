Framework configuration
-----------------------

##### Marker synchronization

- `hConfig.sync_range`
	- search range relative to expected marker position (seconds)
	- default: `[NaN, NaN]`
- `hConfig.sync_smooth`
	- waveform smoothing (seconds)
	- default: `NaN`
- `hConfig.sync_thresh`
	- Mahalanobis distance threshold (number of sigmas)
	- default: `NaN`
